# Submission Checklist

Use this checklist before uploading the project to GitHub and submitting it to the trainer.

- [ ] Project runs successfully
- [ ] README completed
- [ ] Requirements file present
- [ ] .env.example present
- [ ] .gitignore protects .env and .venv
- [ ] Tools tested
- [ ] Custom tool implemented
- [ ] Fallback mechanism implemented
- [ ] Langfuse observability included
- [ ] MCP awareness documented
- [ ] Output file generated
- [ ] Demo video recorded
- [ ] GitHub repository created
